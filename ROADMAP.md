# ROADMAP: Plataforma Integral Clínica Mercè

## 1. Fase de Gestión de Datos (Core CRM) ✅
- [x] **CRUD de Clientes**: Listado completo, fichas médicas con historial de tratamientos y alertas (alergias, enfermedades).
- [x] **Gestión de Servicios**: Catálogo de tratamientos con nombre, duración estimada y precio base.
- [x] **Sistema de Bonos (Vouchers)**: Creación de bonos, asignación a clientes y lógica automatizada para descontar sesiones al finalizar una cita.

## 2. Fase de Agenda y Reservas (Intermedia) 📅
- [x] **Calendario Interactivo**: Vista mensual/semanal en el Dashboard para que Mercè gestione los huecos (drag&drop).
- [ ] **Motor de Reservas (Landing)**: Formulario público conectado a la disponibilidad real, calculando los espacios libres de forma automática.
- [x] **Estados de Cita**: Flujo de estados sincronizado (Pendiente, Confirmada, Realizada, Cancelada, No asistió).

## 3. Fase de Facturación y Documentación ✅
- [x] **Facturación**: Generación automatizada de presupuestos y facturas/recibos en formato PDF descargable con sello interactivo.
- [x] **Consentimientos Informados (RGPD)**: Pantalla táctil adaptada para firmar digitalmente desde una tablet o móvil antes del tratamiento.

## 4. Fase de Seguridad y Pulido (Final) 🛡️
- [x] **Panel de Ajustes**: Gestión de datos de clínica, logos, firma y backups JSON.
- [ ] **Roles y Permisos**: Restricción de áreas. Distinción de vistas entre `admin` (Mercè) y empleados si la clínica crece.
- [x] **Validación y UX Profesional**: Validaciones estrictas y manejo de errores. Cohesión con la paleta de marca (Soft Rose & Gold).
