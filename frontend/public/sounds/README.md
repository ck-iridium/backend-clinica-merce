# Sonidos del Sistema de Notificaciones

## Archivo requerido: `notification.mp3`

Coloca aquí un archivo llamado exactamente `notification.mp3`.

### Dónde conseguirlo (gratis y sin licencias):

**Opción A — Mixkit (recomendado):**
- Visita: https://mixkit.co/free-sound-effects/notification/
- Sugerencias elegantes para clínica:
  - "Correct answer tone" → suave y profesional
  - "Software interface start" → chime de cristal
  - "Notification bells" → con clase
- Descarga en MP3 y renombra a `notification.mp3`

**Opción B — Freesound.org:**
- https://freesound.org/search/?q=soft+chime+notification
- Filtra por licencia CC0 (dominio público)

**Opción C — Zapsplat:**
- https://www.zapsplat.com/sound-effect-category/notifications-and-alerts/
- Requiere registro gratuito

### Especificaciones recomendadas:
- Duración: < 2 segundos
- Formato: MP3
- Estilo: "soft chime", "glass pop" o "gentle ping"
- Evitar: campanadas estridentes, sonidos de alarma

Una vez colocado el archivo aquí, el sistema lo cargará automáticamente.
